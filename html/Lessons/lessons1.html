JAVA BASIC SYNTAX

Java Syntax is a set of rules that governs how Java programs must be written so that the compiler can properly read and interpret them. Following these defined rules is crucial for ensuring that the code is logically correct, readable, and free of grammatical errors. Understanding this syntax is the foundation for writing any Java program, such as a basic "Hello World" application.

CONDITIONAL STATEMENTS:

In Java, conditional statements run specific blocks of code depending on whether a given boolean expression evaluates to true or false.

If

The most basic conditional statement. It executes a block of code only if a specified condition evaluates to true. If the condition is false, the code block is skipped.

else

The optional counterpart to the if statement. The else block of code is executed only when the if condition (and any preceding else if conditions) evaluates to false. It provides a default path of execution.

else if

A way to test multiple conditions sequentially. If the initial if condition is false, the program checks the next else if condition. This process continues until a true condition is found, and its corresponding code block is executed. If a condition is true, the rest of the else if and the final else blocks are skipped.

STRING METHODS:

The Java String class provides several built-in methods that allow programmers to effectively manipulate and query textual data.

These methods include:

.length()

Returns the total number of characters (the length) in the string.

.concat()

Joins the current string with another specified string, creating a new combined string.

LOOPS:

Java uses four primary types of loops to repeatedly execute a block of code based on a boolean condition until that condition evaluates to false.

while Loop

The while loop checks the boolean condition before executing the loop's body. The code inside the loop will execute zero or more times.

do-while Loop

The do-while loop executes the loop's body at least once before checking the boolean condition. The condition is checked only after the first iteration.

for Loop

The for loop is typically used when the number of iterations is known. It combines the initialization, condition check, and iteration update into a single, compact statement.

for-each Loop (Enhanced for Loop)

The for-each loop is designed for easy iteration over arrays and collections. It simplifies the process of stepping through every element without needing to manage index variables.

BREAK AND CONTINUE:

The Java keywords break and continue provide precise control over the flow and iteration of loops.

break Keyword

The break keyword is used to immediately terminate the innermost loop (or switch statement) in which it is executed.

  • When break is encountered, the program exits the loop entirely, and execution resumes with the statement immediately following the loop's closing curly brace.

continue Keyword

The continue keyword is used to skip the current iteration of a loop and immediately proceed to the next iteration.

  • When continue is encountered, any remaining code within the loop body for that iteration is skipped.
  • For a for loop, execution jumps to the update/increment part.
  • For while and do-while loops, execution jumps directly back to the condition check.

These two keywords are primarily used inside conditional blocks (if) within a loop to handle specific conditions.
