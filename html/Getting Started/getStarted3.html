Writing your First Java Program

Create a new project

  1. Launch IntelliJ IDEA.
  2. If the Welcome screen opens, click New Project. Otherwise, go to File | New | Project in the main menu.
  3. From the list on the left, select Java. Give the project a name.
  4. Select Maven as the Build system and check the Add sample code.
  5. From the JDK list, select the latest available Oracle OpenJDK version.
  6. If the JDK is installed on your computer, but not defined in the IDE, select Add JDK and specify the path to the JDK home directory.
  7. If you don't have the necessary JDK on your computer, select Download JDK.
  8. Click Create.

Understanding the Java Hello World Program:

Class Definition

Every Java program must have at least one class. Here, the class is defined using the class keyword:

main Method

In the Java programming language, every application must contain a main method as it is the entry point of the application:

  • public: Allows JVM to access the method from anywhere.
  • static: Method can run without creating an object.
  • void: It doesn’t return any value.
  • String[] args: Accepts command-line arguments.

System.out.println()

This prints output to the console:

  • System: Built-in class from java.lang package.
  • out: Static member (PrintStream object) of System.
  • println(): Method that prints to console and moves to the next line.
