Basic Data Types

Because Java is a statically typed language, it requires variable types to be explicitly declared and known at compile time. This strict requirement allows the compiler to catch type errors early, preventing incorrect assignments like attempting to store a string in an integer variable from ever compiling.

In Java, a data type is a mandatory declaration that defines the structure and size of data a variable is capable of holding. This structure fundamentally determines the storage mechanism:

  • Primitive Types: Variables of these types hold the actual data value within their allocated memory space.
  • Non-Primitive Reference Data Types: Variables of these types hold a pointer to the location in memory (the heap) where the actual object data resides.memory space.

Primitive Vs. Non-primitive/ Reference Data Types

PRIMITIVE DATA TYPES

NON-NUMERIC

Non-numeric data types are classifications of data that do not represent quantities and therefore cannot be used in mathematical calculations (like addition, subtraction, or averaging). Instead, they are used to store textual, logical, or other complex structures.

boolean

Represents one of two logical values: true or false. Commonly used for conditional checks.

char

A 16-bit Unicode character used to store single symbols or letters.

NUMERIC

Numeric data types are classifications of data that store quantitative values, meaning they represent numbers that can be used in mathematical operations, such as addition, subtraction, multiplication, and division.

Integer

Integer are used to store non-decimal, whole numbers (both positive and negative). Java provides four options: byte, short, int, and long. These differ in the amount of memory they occupy and, consequently, the range of numbers they can hold. Selecting the most appropriate type is a decision based on the numeric value's size to ensure memory efficiency.

byte

An 8-bit signed integer used to save memory in large numeric arrays.

short

A 16-bit signed integer often used when memory is limited and values are moderate in size.

int

A 32-bit signed integer commonly used for whole numbers.

long

A 64-bit signed integer used when int is not sufficient for large values.

Floating Point

Floating Point are a class of numeric data types used in programming to store numbers that have a fractional component that is, numbers with a decimal point.

float

A 32-bit single-precision floating-point type used for fractional values.

double

A 64-bit double-precision floating-point type and the default for decimal numbers.

NON-PRIMITIVE DATA TYPES

String

A sequence of characters used to represent textual data in programming, which can include letters, numbers, symbols, and spaces. It is a fundamental data type typically enclosed in quotation marks and is essential for storing and manipulating human-readable information.

Array

Stores multiple elements of the same type in a single structure. Java arrays are objects, dynamically allocated, and indexed from 0.
