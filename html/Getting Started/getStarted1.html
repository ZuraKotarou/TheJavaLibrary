Installing the Java Development Kit (JDK)

Before you can write or run any Java programs, you need to install the Java Development Kit (JDK). The JDK contains all the tools needed to compile and execute Java code, including the Java compiler and the Java Virtual Machine (JVM).

Installing Java also requires setting up the correct environment variables on your system so that Java commands can run smoothly in the command prompt. Below is a clear, step-by-step guide on how to download, install, and configure Java on a Windows 64-bit operating system.

Step 1: Visit the Official Oracle Website

To download Java safely, you should always use the official Oracle website. Go to:oracle.com/java and click on the “Download Java” button. This will take you to the page where you can find the latest version of the Java Development Kit.

Step 2: Download the JDK for Windows

On the download page, you will see the latest JDK version available—typically something like JDK 23. Several download options will appear for different operating systems such as Linux, macOS, and Windows.

Since we are installing on Windows:

  1. Click the Windows tab.
  2. Select the x64 Installer (.exe) option to download the installer for 64-bit Windows
The .exe file will start downloading, and once complete, it will be available in your Downloads folder

Step 3: Install the Java Development Kit (JDK)

Navigate to your Downloads folder and double-click the file named something like: jdk-23_windows-x64_bin.exe
This will launch the Java Installation Wizard.

Click Next to begin the installation.

The installer will show you the default installation path—usually in: C:\Program Files\Java\jdk-23
Click Next again, and the installation process will start.

Once it completes, click Close to finish the installation.

Step 4: Set Up the Environment Variables

To run Java from the command prompt, you must configure the PATH environment variable.

  • Open your C: drive and navigate to: C:\Program Files\Java\jdk-23\bin
  • Copy the full path from the address bar—this is the location of all Java tools.
  • Now search your computer for “Environment Variables” and open it.
  • In the System Properties window, click Environment Variables.
  • Under the System variables section, find and select the variable named Path.
  • Click Edit.
  • Click New, and paste the path you copied earlier.
  • Press OK to save the changes.

Setting this PATH ensures that Windows can locate Java commands like java and javac from any folder in the command prompt.
