Compiling and Running a Java Program

Compiling a Java Program Using Command Prompt vs. IntelliJ IDEA

When learning Java, it is important to understand two common ways to compile and run your programs:

  1. Using the Command Prompt (Windows) or Terminal (Mac/Linux)
  2. Using an Integrated Development Environment (IDE) such as IntelliJ IDEA

Both methods achieve the same goal, but they offer different experiences and levels of control. Understanding both will help you become a more flexible and confident Java developer.

Compiling and Running Java Programs Using Command Prompt

Even though IDEs provide built-in tools for compiling and running Java programs, Java can also be executed directly from the command line. This method gives you a deeper understanding of how Java works behind the scenes.

Save Your Java Program

Use a text editor (e.g., Notepad) or any IDE to write your .java file.

  • The filename must match the class name containing the main() method.
  • Save it with the extension .java (e.g., Main.java).
  • Remember the folder where you saved the file.

Open Command Prompt or Terminal

  • Windows: Open Start Menu → Type CMD → Press Enter.
  • Mac: Finder → Go → Applications → Utilities → Terminal.

Compiling and Running Java Programs Using IntelliJ IDEA

IntelliJ IDEA simplifies the entire process of writing, compiling, and running Java programs. It performs all compilation steps behind the scenes with a single click.

Run the Application

To run your program:
  • Click the green play gutter icon beside the main() method
  • Select Run ‘Main.main()’

IntelliJ compiles your code automatically—no need to type javac or java.

View Output

The result appears in the Run tool window at the bottom.

If the program executes successfully, you’ll see:

Process finished with exit code 0

which means no errors occurred.
